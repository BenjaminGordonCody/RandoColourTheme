Rando: A Colour Theme Generator

This is a webpage for generating random RGB colour themes for websites. If you click any of the elements in this text-box (for eg, the title, the paragraphs, the two images) you can then recolour them by clicking a swatch in the opposite box. If you'd like to select the background, then click the button labelled 'bg'.

If you'd like a new, randomly generated pallette of colours to play with, click the "New Swatches" button. These In the bottom left of the screen is a record of which colours you've selected. These won't change if you generate new swatches.
